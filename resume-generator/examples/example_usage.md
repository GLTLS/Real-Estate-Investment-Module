# 简历生成技能使用示例

## 示例1: 基本使用

### 用户输入
```
/resume-generator "招聘数据分析专家，要求5年以上金融行业数据分析经验，熟练使用Python和SQL，有Tableau或Power BI经验者优先，需要良好的英语沟通能力。"
```

### 技能响应
1. 检查母版文件是否存在 ✓
2. 分析职位要求:
   - 5年以上金融行业数据分析经验
   - Python和SQL技能
   - Tableau/Power BI经验
   - 英语沟通能力
3. 生成优化的简历
4. 保存到: `C:\Users\78724\Documents\AI简历\RESUME_2026-04-08T11-45-30\`
5. 打开所有生成的文件

## 示例2: 更新母版后生成

### 用户输入
```
/resume-generator "寻找金融科技数据分析主管，要求8年以上经验，CFA认证，熟悉云计算平台，有团队管理经验。" --update-master
```

### 技能响应
1. 打开母版文件供编辑
2. 等待用户更新信息
3. 用户编辑完成后回复"继续"
4. 分析职位要求:
   - 8年以上经验
   - CFA认证
   - 云计算平台经验
   - 团队管理经验
5. 生成针对金融科技主管的简历
6. 突出显示CFA认证和团队管理经验

## 生成的简历内容

### 英文HTML简历包含
- 专业头部信息（姓名、标题、头像）
- 职位匹配说明
- 专业概述（针对职位优化）
- 技术技能（突出相关技能）
- 工作经历（量化成就）
- 项目经历（相关项目）
- 专业认证
- 教育背景和语言能力
- 量化成就统计

### 中文HTML简历包含
- 相同内容的中文版本
- 相同设计和布局
- 中文优化表述

### 文本简历包含
- 简洁格式
- 关键信息
- 适合申请系统粘贴

## 简历优化特点

### 针对职位优化
- 根据职位要求调整技能展示顺序
- 突出相关经验和认证
- 使用职位相关关键词
- 强调匹配的能力

### 量化成果展示
- 使用具体数字展示成就
- 突出效率提升百分比
- 显示团队规模和管理经验
- 展示项目成果影响

### 专业设计
- HSBC品牌风格（金融行业）
- 响应式布局
- 清晰的视觉层次
- 重点内容突出
- 适合打印的格式

## 使用建议

### 最佳实践
1. **提供详细的职位描述** - 越详细，优化效果越好
2. **定期更新母版** - 保持信息最新
3. **使用有意义的文件夹名** - 方便管理不同职位的简历
4. **查看生成的简历** - 确保内容准确无误
5. **根据反馈调整** - 根据申请结果优化母版

### 常见职位类型
- **金融科技数据分析** - 强调Python、SQL、金融知识
- **风险管理** - 突出FRM、风险建模、合规经验
- **投资管理** - 强调CFA、投资分析、组合管理
- **商业智能** - 突出Power BI、Tableau、数据可视化
- **数据工程** - 强调ETL、数据管道、云计算

### 母版维护
- 定期更新工作经历
- 添加新完成的项目
- 更新技能和认证
- 保持联系信息最新
- 记录量化成就
